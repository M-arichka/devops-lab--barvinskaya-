## Описание проекта
Данный репозиторий создан для выполнения лабораторных работ по курсу "Введение в веб технологии". Здесь будут храниться отчеты, код и другие материалы, связанные с изучением веб-технологий и DevOps практик.

## План изучения DevOps
1. **Основы Git и GitHub**
   - Настройка репозитория
   - Работа с ветками
   - Создание Pull Request

2. **Основы веб-технологий**
   - HTML/CSS верстка
   - JavaScript программирование
   - Работа с backend

3. **DevOps практики**
   - CI/CD pipelines
   - Контейнеризация с Docker
   - Деплой приложений

## Структура репозитория
- `/lab0` - лабораторная работа №0 (настройка окружения)
- `/lab1` - лабораторная работа №1
- `/lab2` - лабораторная работа №2
- `/lab3` - лабораторная работа №3
